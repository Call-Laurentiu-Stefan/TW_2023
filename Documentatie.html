2.Descriere generala

2.1 Descrierea Produsului

Aplicatia European Obesity Tracker, bazata pe proiectul Obesity Visualizer, este developata pentru persoanele care vor sa caute informatii despre obezitatea dintr-o anumita tara din zona Europei, in functie de necesitatile acestora.

2.2 Functiile Produsului

Aplicatia web managerieaza informatiile despre obezitatea dintr-o tara ce face parte din zona Europei. Despre aceste tari sunt informatii precum : anul in care s-a facut masuratoarea pentru obezitate, valoarea propriu zisa ce reprezinta rata obezitatii din anul respectiv, vizualizarea datelor folosind Table, Line si Bar, sortarea in ordine descrescatoare a tarilor in functie de BMI(Body Mass Index), exportarea charturilor in unul din formatele: CSV, WEBP, SVGsi selectarea unei singure tari asupra careia ii se vor aplica filtre

Index page : Pagina din care pot fi accesate celelalte pagini ale website-ului

  • login : redirectioneaza catre login page.
  • contact : redirectioneaza catre contact page.
  • about : redirectioneaza catre about page.
Login page :
  • log in : valideaza logarea utilizatorului si il redirectioneaza catre index page
Admin page :
  • login : redirectioneaza catre login page.

2.3 Clasele și caracteristicile utilizatorilor

  • User : user asociat cu un cont poate utiliza facilitatile website-ului.
  • Admin : are optiunea de a manageria aspecte diferite ale website-ului.

2.4 Mediul de operare

Pagina web poate fi utilizata pe orice device care are o conexiune la internet si un browser instalat.

2.5 Constrângeri de proiectare și implementare

European Obesity Tracker va fi dezvoltat in Php, si va folosi html, css si javascript pentru interfata.

4. System Features

4.1 Index page

Oaspeții și utilizatorii pot naviga la toate celelalte pagini de pe această pagină. De asemenea, utilizatorii pot cauta diferite tari asupra carora se aplica filtre si are posibilitatea de a vizualiza rezultatul sub forma unui chart la alegere.

4.2 Login page

Utilizatorii aplicației se pot autentifica introducând informatiile corecte și, dupa validare, accesează alte funcționalități ale aplicației.

4.3 Admin page

Utilizatorul cu rol de admin al aplicației poate sa insereze si sa modifice informatii existente

4.4 About page

Utilizatori conectati sau nu pot observa informatiile despre pagina noastra

4.5 Contact page

Utilizatori conectati sau nu pot observa informatii despre dezvoltatorii aplicatiei

5. Cerinte Nonfunctionale

5.1 Safety Requirements

Pentru a se asigura că niciunul dintre utilizatorii European Obesity Tracker nu pierde date în timpul utilizării aplicației, echipa de dezvoltatori lucrează în mod regulat la European Obesity Tracker.

5.2 Cerinte de securitate

European Obesity Tracker nu are cerințe de securitate și, prin urmare, orice tip de utilizator îl poate folosi fără privilegii suplimentare.
