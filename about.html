Welcome to the Obesity Statistics Web App

This web application is capable of visualizing and comparing information on obesity rates in populations, using public data provided by Eurostat via a REST/GraphQL API. You can explore statistics, comparisons based on multiple criteria, and generate at least three types of visualizations. These can be exported in CSV, WebP, and SVG formats.

Features

  • Visualize and compare obesity rates in populations
  • Use public data from Eurostat via a REST/GraphQL API
  • Explore statistics and comparisons based on multiple criteria
  • Generate at least three types of visualizations
  • Export visualizations in CSV, WebP, and SVG formats

Technologies Used

  • HTML
  • CSS
  • PHP
  • REST/GraphQL API
  • CSV, WebP, and SVG formats
